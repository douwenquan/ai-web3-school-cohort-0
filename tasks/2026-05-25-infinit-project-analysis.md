## 任务描述

拆解 AI × Web3 方向的一个代表性项目：INFINIT — AI 驱动的 DeFi 执行层。

## 项目全貌

**INFINIT** 是由 INFINIT Labs 打造的 AI 驱动 DeFi 执行层，将复杂的多步骤 DeFi 策略抽象为一键式操作，实现 "提示即 DeFi"（Prompt-to-DeFi）的端到端自动化。

---

## 一、项目背景

### 核心定位：DeFi 的"意图层"

用户无需理解底层协议技术细节，只需用自然语言描述意图，Agent Swarm 自动完成协议选择、策略构建、跨链桥接、Gas 优化与交易执行的完整闭环。

### 核心价值主张

| 特性 | 说明 |
|------|------|
| **零代码策略创建** | 自然语言描述目标（如"做多 ETH 做空 BTC 基差交易"），Agent 自动执行 |
| **非托管安全模型** | 用户连接 EVM 钱包，设置可配置支出限额和期限，交易需用户签名确认 |
| **策略知识变现** | 策略创作者将专业知识转为可执行策略，获得手续费分成 |

## 三、产品矩阵

### 3.1 INFINIT Intelligence

AI 核心引擎，三大职能：
- **个性化策略推荐** — 分析钱包资产和历史行为，结合市场数据推荐策略
- **跨链自动执行** — Agent Swarm 协调多链操作，自动化桥接和 Gas 管理
- **智能交互助手** — 基于 LLM 的对话式 DeFi 操作

技术路线：ReAct（推理-行动）+ RAG（检索增强生成）+ 用户偏好学习

### 3.2 Prompt-to-DeFi

用户用自然语言描述意图，系统自动完成策略构建、验证与执行。从 "我应该拿 ETH 做什么？" 到 "策略已成功执行" 的连续对话流。

### 3.3 INFINIT Strategies

确定性策略组件库。所有策略为确定性编译代码，消除 LLM "幻觉" 在执行层的风险。策略上架前经过模拟测试，用户可预览每一步操作。

### 3.4 Universal Wallet Integration

将 Agentic DeFi 能力嵌入主流 Web3 和 CEX 钱包界面。

## 四、技术架构

### 4.1 Agent Swarm（35+ 专业代理）

| 代理类型 | 说明 |
|----------|------|
| **协议专用代理** | 每个代理绑定特定协议（Aave、Pendle、Venus、Uniswap 等） |
| **功能专用代理** | Swap、Bridge、LP、Stake 等通用操作 |
| **智能编排** | 根据用户请求自动选择最优代理组合 |

**多代理协作示例**：
> 用户意图："在 Aave 存款后到 Pendle 购买 PT 代币"
> → Aave 代理分析最优收益机会
> → Pendle 代理执行 PT 代币购买
> → 如有需要 Hyperliquid 代理开设对冲仓位

### 4.2 三层安全防护

| 层 | 机制 |
|----|------|
| **1️⃣ 确定性代码编译** | LLM 仅用于意图翻译，实际执行是确定性代码，消除执行层幻觉 |
| **2️⃣ 内部策略验证** | 每个 AI 生成策略上架前经过后台模拟测试 |
| **3️⃣ 执行前模拟 + 用户审批** | 用户可预览完整执行路径，签署确认后才执行 |

### 4.3 可审计性

接入 EigenCloud 审计基础设施，每步操作（提示词处理 → 模型运行 → 数据处理）均留下可审计痕迹，实现全链路透明。

## 八、优势与挑战

### 优势 ✅
- **产品化领先**：17.4 万用户 + 45 万交易，最丰富的真实反馈
- **多代理协作深度**：35+ 代理的 Swarm 架构，跨协议策略复杂度高
- **知识变现飞轮**：策略创作者经济，可形成网络效应
- **机构背书**：Electric Capital、Hashed、Maelstrom 等顶级投资

### 挑战 ⚠️
- **代币价格承压**：较高点下跌 78%，二级市场信心待修复
- **用户地域集中**：核心用户主要在亚洲，欧美市场待拓展
- **无 ZK 能力**：相比引入 ZK 的项目在隐私保护上存在差距
- **执行可靠性**：代理数量和策略复杂度增长带来稳定性考验
