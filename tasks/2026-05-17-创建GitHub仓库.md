# Week 1｜前置准备｜创建课程 GitHub Repo

## 任务描述

创建一个用于记录 AI x Web3 School 学习过程的 GitHub repo。建议优先让 Learning Agent 协助初始化和维护这个仓库，而不是只创建一个空 repo。

## 任务目标

- 创建个人学习仓库
- 初始化仓库结构

## 执行过程

1. 使用 GitHub CLI 创建仓库：`ai-web3-school-cohort-0`
2. 设置仓库为 public
3. 使用 Learning Agent（OpenClaw/Hermes）初始化仓库结构
4. 创建目录：daily/, tasks/, experiments/, handbook-feedback/, hackathon/, submissions/
5. 创建模板文件：daily-note.md, task-note.md
6. 编写 README.md, profile.md, learning-plan.md
