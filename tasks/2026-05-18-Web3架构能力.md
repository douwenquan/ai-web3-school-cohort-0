## 实际结果

**关键信息**:
- Web3 安全：基本概念和常见风险点
- Web 2 vs Web 3 核心区别：中心化可回滚 vs 链上不可逆
- AI 时代 Web3 需要更强架构能力，门槛提升

**学习材料**:
- Updraft Career Tracks - 各方向学习路径
- GMWalletApp/epusdt - 多链支付网关项目

**个人理解**:
- Web 2 是中心化系统，大部分操作、交易可以回滚或冻结
- Web 3 是分布式系统，大部分操作在链上，尤其是 DEX 交易所
- 在 AI 时代，Web 3 需要更强的架构能力
- 这也变相提升了人的能力门槛
