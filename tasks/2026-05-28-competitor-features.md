## 第一类：投资组合追踪器

### DeBank

**一句话定位：** Web3 原生钱包 + 最佳 Web3 投资组合追踪器

**官宣卖点：**
- 覆盖所有 EVM 链的 token、DeFi 协议、NFT
- 自带 Web3 原生通讯（DeBank Messenger）
- 实时追踪 DeFi 头寸、借贷仓位、奖励
- 多链资产聚合视图

**核心功能：**
- 多链资产余额展示（EVM 链全覆盖）
- DeFi 头寸详情（LP、借贷、Staking）
- NFT 组合查看
- 代币价格走势
- 链上交易历史
- 通迅功能（DeBank Messenger）

**覆盖范围：** 30+ EVM 兼容链

---

### Zapper

**一句话定位：** 一站式 Web3 资产管理平台

**官宣卖点：**
- 最简洁的 DeFi 组合管理仪表盘
- 不仅看资产，还能直接操作（存入/提取/兑换）
- 社区驱动的 NFT 画廊

**核心功能：**
- 多链资产聚合
- DeFi 仓位管理（一键存入/提取）
- 交易功能（聚合报价）
- NFT 管理
- 社区积分/成就系统

**覆盖范围：** 多链（EVM + 部分非 EVM）

---

### Zerion

**一句话定位：** 加密钱包 + DeFi 组合追踪器 + 交易聚合器

**官宣卖点：**
- 内置钱包，从追踪到交易一站式完成
- 自动发现所有 DeFi 仓位（staked tokens、LP、rewards、debts）
- 智能路由报价（聚合 DEX）
- 支持 Ledger 硬件钱包

**核心功能：**
- 组合追踪（自动发现所有头寸）
- 内置钱包（收发资产）
- 交易聚合（最优报价路由）
- NFT 追踪
- 多链支持
- 硬件钱包兼容

**覆盖范围：** 多链

- 从信号到交易执行（Nansen AI）
- 机构级链上数据

**核心功能：**

## 竞品定位总结对比

从三个维度看这些竞品：

**用户是谁：**
- DeBank/Zapper/Zerion → 普通 DeFi 用户，想知道自己有多少资产
- Nansen → 专业交易者/机构，想知道聪明钱在干什么
- Arkham → 调查员/研究员，想知道地址背后的实体
- Dune → 数据分析师，想自己查链上数据
- Chainalysis/Elliptic/TRM → 政府/交易所/合规部门，想查违法活动

**解决的问题：**
- DeBank/Zapper/Zerion → 「我持有多少？」（资产追踪）
- Nansen → 「谁在赚钱？」（聪明钱追踪）
- Arkham → 「这是谁的地址？」（实体关联）
- Dune → 「链上发生了什么？」（数据查询）
- Chainalysis/Elliptic/TRM → 「有没有洗钱？」（合规）

**不做的事：**
- 没有人做：「我的交易行为怎么样？」（行为分析+诊断+建议）
- 没有人做：生成个人行为画像
- 没有人做：基于历史交易给出改善建议
