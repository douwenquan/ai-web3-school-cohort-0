# Handbook Feedback

本目录用于记录学习过程中发现的问题、卡点和建议，反馈给 Handbook 维护团队。

## 反馈类型

- ❓ **概念不清**：某个概念解释不清楚或缺少背景知识
- 🐛 **内容错误**：信息有误或链接失效
- ✏️ **错别字**：拼写或语法错误
- 📝 **建议**：结构改进、内容补充、案例增加
- ⏰ **资料过期**：内容需要更新

## 反馈格式

每条 feedback 单独一个文件，命名格式：`YYYY-MM-DD-description.md`

## Feedback 模板

```markdown
## 页面链接

（Handbook 中的具体页面 URL）

## 问题描述

（描述你遇到的问题或建议）

## 建议改法

（如果可以，提出具体的修改建议）

## 来源

（哪节课/哪天的学习遇到的）
```

## 反馈提交

通过 GitHub PR 提交到：https://github.com/lxdao-official/aiweb3school
