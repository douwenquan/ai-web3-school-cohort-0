## 二、Week 1 学完了什么

### AI 基础（5/19 + 5/22）

| 概念 | 掌握程度 |
|------|---------|
| LLM | Token、Embedding、Transformer、Hallucination、Multimodal |
| Prompt | Prompt Engineering、System/User Prompt、Prompt Injection 安全 |
| Context | Context Window、Memory、上下文压缩策略 |
| RAG | Chunking → Embedding → Vector DB → Retriever → Rerank → Citation 完整流程 |
| Agent | 模型+工具+Harness、Plan/Reflection、Multi-Agent |
| Frameworks | LangChain、LangGraph、OpenAI Agents SDK、DSPy、Hermes 对比 |
| MCP | Server/Client、Tool Schema、Permission |
| Evaluation | 评估集设计（30 条样本）、回归测试 |
| Fine-tuning | SFT、LoRA、PEFT、Overfitting |
| Inference | 托管 API vs 本地部署、量化技术 |
| VibeCoding | Claude Code、Codex、OpenCode、适用场景与边界 |
| **最小实践** | 11 项：交易解释器、风险摘要 Prompt、钱包授权检查 Agent、协议文档 RAG、DAO 提案 Agent、框架对比、VibeCoding 闭环、MCP Server、最小 Eval、微调前评估、推理对比 |

### Web3 基础（5/18 + 5/20 + 5/23 + 5/24）

| 章节 | 核心概念 |
|------|---------|
| **Cryptography** | Hash、公私钥、签名、Merkle Tree |
| **Wallet / Account** | EOA / CA / 多签、助记词、交易生命周期、Gas（EIP-1559）、L2 节省策略、区块浏览器 |
| **Smart Contract** | Solidity、EVM、ABI、Event、Upgrade（Proxy/Diamond）、安全要点 |
| **Dev Workflow** | Remix → Hardhat/Foundry → OpenZeppelin → Web3Modal/Wagmi |
