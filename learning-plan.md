## 学习路径建议

基于你的背景（AI 有基础、Web3 需加强、开发能力强），建议按以下顺序推进：

### 第一阶段：补齐 Web3 基础（约 2 周） ✅ 已完成 (2026-05-25)

优先阅读：
1. [Network](https://aiweb3.school/zh/handbook/web3/network/) - 区块、共识、L2、RPC
2. [Cryptography](https://aiweb3.school/zh/handbook/web3/cryptography/) - 哈希、公私钥、签名
3. [Wallet](https://aiweb3.school/zh/handbook/web3/wallet/) - 钱包、身份、签名入口
4. [Smart Contract](https://aiweb3.school/zh/handbook/web3/smart-contract/) - 合约部署、调用、状态
5. [Account Abstraction](https://aiweb3.school/zh/handbook/web3/account-abstraction/) - Smart Account

### 第二阶段：AI × Web3 Bridge（约 2-3 周）

核心章节：
1. [Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/) - 链上状态进入 Agent 上下文
2. [Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/) - RPC、钱包、合约工具调用
3. [Agent Workflow](https://aiweb3.school/zh/handbook/bridge/agent-workflow/) - 自动化 vs human-in-the-loop
4. [Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/) - Agent 权限管理
5. [Machine Payment](https://aiweb3.school/zh/handbook/bridge/machine-payment/) - 机器间支付结算

### 第三阶段：前沿探索与 Hackathon（约 3-4 周）

根据兴趣选择 Track：
- [Agentic Commerce](https://aiweb3.school/zh/handbook/tracks/agentic-commerce/) - Agent 发现、协商、支付、凭证
- [Wallet / Permission](https://aiweb3.school/zh/handbook/tracks/wallet-permission/) - 钱包、Session Key、Policy
- [Dev Tooling](https://aiweb3.school/zh/handbook/tracks/dev-tooling/) - 合约理解、测试、代码审查
